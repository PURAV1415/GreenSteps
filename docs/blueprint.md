# **App Name**: EcoTrack

## Core Features:

- User Authentication: Secure signup and login functionality for users, capturing name, email, department, campus, and transport details.
- Daily Transport Input: Allow users to input their daily transport mode, distance, and trips, updating emissions and points in real-time.
- Carbon Emission Tracking: Calculate and display total and daily carbon emissions based on user transport data, with charts for visualization.
- Points and Reward System: Implement a points system with streaks, milestones, and personalized suggestions to incentivize sustainable transport choices.
- Personalized Eco-Friendly Recommendations: AI powered tool that uses the LLM tool to evaluate daily carbon footprint and mode of transportation used, and generate personalized prompts that yield a high reward in carbon point reduction when implemented.
- Leaderboard: Show department and campus-level leaderboards, ranking users by total and daily points, promoting competition.
- Data Storage: Store all user and carbon emission tracking data on Firestore to support updating emissions, point scoring, and generation of the leaderboard

## Style Guidelines:

- Primary color: A vibrant, natural green (#90EE90) to represent sustainability and growth.
- Background color: A light, desaturated off-white (#F5F5DC) for a clean and accessible interface.
- Accent color: An analogous blue (#ADD8E6) to the primary color for interactive elements and points displays.
- Headline font: 'Space Grotesk' (sans-serif) for a tech-friendly and modern feel; body text: 'Inter' (sans-serif).
- Use eco-friendly icons and animations to represent carbon emissions and sustainable transport.
- A clear and intuitive layout with distinct sections for profile, emissions, points, and leaderboards.
- Subtle animations for updating points, emissions charts, and leaderboard transitions.
